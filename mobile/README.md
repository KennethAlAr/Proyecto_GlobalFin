# Aura Wellness Mobile App (Flutter)

This directory contains the Flutter project for the Aura Wellness mobile application.

## Setup

1. Install Flutter SDK: https://flutter.dev/docs/get-started/install
2. From this folder, run:
   ```bash
   flutter pub get
   flutter run
   ```
3. The `lib/main.dart` file is the entry point. Scaffold new features under `lib/`.

> **Note:** If this is an empty placeholder, you may also run `flutter create .` in this directory to generate the full project structure.

## Project Structure

- `lib/` - Dart source files
- `test/` - Unit and widget tests
- `pubspec.yaml` - Dependencies and asset configuration
